### Integration Issues

**INFRA-03 fragility:** The CI build script (`npm run build`) triggers `build:data` which makes a live HTTP POST to ecdysis.org. This is a non-deterministic failure point: if ecdysis.org is unavailable, all CI builds fail on all branches. The committed `frontend/src/assets/ecdysis.parquet` is a valid fallback. This is acknowledged tech debt from Phase 1; the code is correct but the CI dependency is fragile.

**PIPE-01 documentation:** Phase 1 SUMMARY and milestone requirement description reference `--db 164`, but the actual CLI flag is `--datasetid` (code correct, docs stale).
