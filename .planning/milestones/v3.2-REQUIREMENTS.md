## Out of Scope (v3.2 hard locks тАФ see archive for full table)

| Feature | Reason |
|---------|--------|
| Attribute filters (eye color, ID character, ease of photo ID) | Seed lock тАФ deserves its own design milestone |
| Species detail pages | Seed lock тАФ cards are self-contained surfaces in v3.2 |
| Identification / dichotomous key UI | Out of scope per seed |
| Build-time iNat photo fetch | Seed lock тАФ flaky CI and rate-limit risk |
| Headless-browser SVG capture | Seed lock тАФ uses Python xml.etree instead |
| Cross-table joins to floral hosts | Seed feature creep |
| Per-volunteer custom checklists | Cold-start risk |
| Photo upload UI on live site | Cold-start; community feature territory |
| Embedding the SPA map inside a card | Performance death; deep-link is the right affordance |
| Manual species-list editing bypassing the checklist | Diverges from authoritative source |
