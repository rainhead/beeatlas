## Overview

Ingested a periodic iNaturalist CSV export of 45,354 expert-identified bee observations, deduplicating against 821 Ecdysis-linked records to produce 44,534 net-new rows. Extended the unified occurrence model with a third source arm (`'inat_obs'`), rendering amber points on the Mapbox map, adding a unified source-selection filter, and showing per-source counts on species pages. A `photos.json` artifact stores CC-licensed image URLs per species for future carousel use.

## Milestone Summary

**Key Decisions:**
- iNat expert observations sourced from periodic CSV export (not live API); `ident_user_id` list as quality gate
- 45,354 observations in first export (2011–2026); 821 overlap with Ecdysis `specimen_observation_id` → excluded
- `quality_grade=any` in export (research, needs_id, casual) — expert identification is the quality gate
- Third source type designated `'inat_obs'` (alongside `'ecdysis'` and `'waba_sample'`)
- iNat obs merge into `occurrences.parquet` via `int_combined` ARM 3 (not separate `inat_obs.parquet`)
- dbt contract expands from 31 to 36 columns with `source` + iNat-specific nullable columns
- `src=` encodes *visible* sources; absent = all sources on (inverted from initial plan)
- Checklist records merged into unified Sources filter row — no separate row
- Photo carousel deferred to a dedicated future milestone; `photos.json` stores CC-licensed URLs

**Issues Resolved:**
- `src=` URL param polarity bug caught during UAT and corrected before approval
- `inat_obs:` prefix missing from `o=` param allowlist — silently dropped deep-links to iNat obs; fixed
- `INTEGER[12]` CASE bug in `int_species_universe.sql` fixed in Phase 118-03

**Issues Deferred:**
- Photo carousel UI (`SPE-F01`) — `photos.json` persisted for future use
- Automated iNat CSV refresh (`PIPE-F01`) — manual periodic export is the v4.2 design
- Floral host taxonomy resolution (`PIPE-F02`)

**Technical Debt Incurred:**
- Manual iNat CSV export requires periodic operator action; no automation path exists yet
- `data/raw/inat_expert_obs.csv` is committed to git (1.9MB); will grow with future exports
