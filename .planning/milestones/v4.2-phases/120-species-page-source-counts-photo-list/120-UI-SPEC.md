## Copywriting Contract

All copy changes are in Nunjucks templates. Source: CONTEXT.md decisions D-01, D-03, D-04, D-11.

| Element | Copy | Template Location |
|---------|------|-------------------|
| Species-detail metadata line | `{{ sp.specimen_count }} specimens · {{ sp.inat_obs_count }} community observations` | `species-detail.njk` line 41, replaces `{{ sp.occurrence_count }} records` |
| Species-detail metadata continuation | `· {{ sp.county_count }} counties · {{ sp.ecoregion_count }} ecoregions` | Unchanged; appended after count |
| Genus/subgenus per-species count (occurrence branch) | `{{ sp.specimen_count }} specimens · {{ sp.inat_obs_count }} community observations` | `genus.njk` line 28, `subgenus.njk` analogous line; replaces `{{ sp.occurrence_count }} records` |
| Genus/subgenus per-species count (checklist-only branch) | `{{ sp.checklist_count }} checklist records` | Unchanged |
| Genus/subgenus per-species count (zero branch) | `0 records` | Unchanged |
| Tribe per-genus count | `{{ g.specimen_count }} specimens · {{ g.inat_obs_count }} community observations` | `tribe.njk` line 26; replaces `{{ g.occurrence_count }} records` |
| Atlas link | `View {{ sp.occurrence_count + sp.inat_obs_count }} records on the atlas →` | `species-detail.njk` line 46; replaces "occurrences" with "records" and adds inat_obs_count |
| Empty state (no photo) | `No photo available` | Unchanged; already rendered by `.photo-placeholder` div |
| Error state | Not applicable — static site, no dynamic data fetching in templates |
| Destructive confirmation | Not applicable — no destructive actions |

Primary CTA: "View N records on the atlas →" (D-11; "records" not "occurrences" since N spans all three sources).

## Component Inventory

| Element | Existing class / element | Change |
|---------|--------------------------|--------|
