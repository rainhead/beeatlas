# Milestone v3.6: Simpler Species Index

**Status:** ✅ SHIPPED 2026-05-16
**Phases:** 92–96
**Total Plans:** 13

## Overview

Replaced the monolithic `/species/` all-cards page with a proper per-taxon page architecture. The pipeline now generates hierarchical `Genus/specificEpithet` slugs; multi-color SVG occurrence maps are produced for genera, subgenera, and tribes; Eleventy generates 527 species pages, 42 genus pages, 103 subgenus pages, and 19 tribe pages as static HTML. A searchable family→genus index replaces the old tree-nav + card wall as the `/species/` entry point.

## Milestone Summary

**Key Decisions:**
- Hierarchical slug `Genus/specificEpithet` (not flat `genus-epithet`): case-preserving, path-component-friendly, supports hyphens in epithets
- D-01/D-02 alphabetical `canonical_name` sort as the one true color index: binds Python SVG colors to JS swatch colors; templates must not re-sort
- `occurrences.parquet` dbt mart (not raw Ecdysis table) as occurrence data source for group maps — includes both data arms
- Full-rebuild pipeline (keep, not incremental): incremental dbt materialization rejected in Phase 87 (~3% wall-clock savings, external marts can't benefit)
- Eleventy pagination over custom data lists (`speciesList`, `genusList`, `subgenusList`, `tribeList`): no pagination plugin; `size:1` over each list
- `hslToHex` local function in `_data/species.js` (not exported): Eleventy data-cascade constraint; named exports break the cascade

**Issues Resolved:**
- BLOCKER-01: `nightly.sh` never uploaded `species-maps/` to S3; fixed by adding `aws s3 cp --recursive` line (commit `e9c3eed`)
- `geometry_wkt` → `geom` column rename in county DuckDB source (quick task 260514-fp3 side effect)
- 106 non-bee orphan TOML entries from `content/species-photos.toml` (accumulated over v3.2 seeding)
- Hyphenated epithet false positive in old-slug detection (`LIKE '%-%'` → `NOT LIKE '%/%'`)

**Issues Deferred:**

- Phase 94 human-verify deferred items: photo CSS hero rule, seasonality chart browser render, swatch-to-SVG D-02 color cross-check, mobile responsive layout
