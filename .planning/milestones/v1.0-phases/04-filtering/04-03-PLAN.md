  <how-to-verify>
Open http://localhost:5173 in a browser. Verify each behavior in order:

**1. Sidebar layout**
- Filter controls appear at the very top of the sidebar (taxon input, year inputs, month checkboxes)
- Summary stats appear below filter controls
- Specimen listing (if any) appears below summary stats

**2. Taxon filter**
- Click the taxon input and start typing a family name (e.g., "Apidae") — the datalist dropdown should show matching suggestions labeled by rank (e.g., "Apidae (family)")
- Select "Apidae (family)" — non-Apidae clusters should dim to grey low-opacity; Apidae clusters render at full color
- The summary stats section should switch to "X of Y" format (e.g., "Families: 1 of 16 · Specimens: N of 51,633")
- Clear the taxon input — all clusters return to full opacity; stats show plain totals

**3. Genus filter**
- Type a genus (e.g., "Bombus") — select "Bombus (genus)" from the dropdown
- Only Bombus specimens should be at full opacity; all others ghosted
- Click a full-opacity cluster — sidebar listing should show only Bombus specimens
- Click a ghosted cluster — sidebar should NOT update (listing stays the same or stays closed)

**4. Year range filter**
- Clear taxon filter first. Enter "2024" in the From year input and press Tab or Enter
- Only specimens from 2024 onward should be at full opacity
- Clear the From year and enter "2023" in the To year — only specimens up to 2023 should be at full opacity
- Enter both From and To — only specimens in that range

**5. Month filter**
- Clear year filters. Check "June" and "July" checkboxes
- Only June and July specimens should be at full opacity; all others ghosted
- Uncheck both — all clusters return to full opacity

**6. Combined filter (AND logic)**
- Select a taxon AND check a month — only specimens matching BOTH should be at full opacity
- Filtered specimen count in sidebar should reflect the AND combination

**7. Clear filters button**
- Clicking "Clear filters" (or equivalent UI) resets all controls and restores all clusters

Confirm or describe any issues found.
