**`export_ecdysis_parquet(con)` function (EXP-01, EXP-02):**
SQL query using CTEs:
1. `wa_counties` CTE: `SELECT name AS county, geometry_wkt FROM geographies.us_counties WHERE state_fips='53'`
2. `wa_eco` CTE: `SELECT name AS ecoregion_l3, geometry_wkt FROM geographies.ecoregions WHERE ST_Intersects(ST_GeomFromText(geometry_wkt), (SELECT ST_GeomFromText(geometry_wkt) FROM geographies.us_states WHERE abbreviation='WA'))`
3. `occ` CTE: `SELECT *, ST_Point(CAST(decimal_longitude AS DOUBLE), CAST(decimal_latitude AS DOUBLE)) AS pt FROM ecdysis_data.occurrences WHERE decimal_latitude IS NOT NULL AND decimal_latitude != ''`
4. `with_county` CTE: LEFT JOIN occ to wa_counties via `ST_Within(occ.pt, ST_GeomFromText(c.geometry_wkt))`
5. `county_fallback` CTE: For rows where county IS NULL, use correlated subquery: `(SELECT county FROM wa_counties ORDER BY ST_Distance(ST_GeomFromText(geometry_wkt), (SELECT pt FROM occ o2 WHERE o2.occurrence_id = with_county.occurrence_id)) LIMIT 1)`
6. `final_county` CTE: UNION ALL of with_county (non-null) and county_fallback
7. Same pattern for ecoregion: `with_eco`, `eco_fallback`, `final_eco`
8. Final SELECT producing exactly these 15 columns in this order:
   - `CAST(o.id AS INTEGER) AS ecdysis_id`
   - `o.occurrence_id AS occurrenceID`
   - `CAST(o.decimal_longitude AS DOUBLE) AS longitude`
   - `CAST(o.decimal_latitude AS DOUBLE) AS latitude`
   - `CAST(o.year AS INTEGER) AS year`
   - `CAST(o.month AS INTEGER) AS month`
   - `o.scientific_name AS scientificName`
   - `o.recorded_by AS recordedBy`
   - `o.field_number AS fieldNumber`
   - `o.genus`
   - `o.family`
   - `NULLIF(regexp_extract(o.associated_taxa, 'host:"([^"]+)"', 1), '') AS floralHost`
   - `fc.county`
   - `fe.ecoregion_l3`
   - `links.inat_observation_id`
9. JOINs: `FROM ecdysis_data.occurrences o JOIN final_county fc ON fc.occurrence_id = o.occurrence_id JOIN final_eco fe ON fe.occurrence_id = o.occurrence_id LEFT JOIN ecdysis_data.occurrence_links links ON links.occurrence_id = o.occurrence_id WHERE o.decimal_latitude IS NOT NULL AND o.decimal_latitude != ''`
10. Wrap in `COPY (...) TO '{out}' (FORMAT PARQUET)`
11. After COPY, run a verification query: `SELECT COUNT(*) AS total, SUM(CASE WHEN county IS NULL THEN 1 ELSE 0 END) AS null_county, SUM(CASE WHEN ecoregion_l3 IS NULL THEN 1 ELSE 0 END) AS null_eco FROM '{out}'` — print these counts to stdout. Assert null_county == 0 and null_eco == 0.

**`export_samples_parquet(con)` function (EXP-03):**
SQL query using CTEs:
1. Same `wa_counties` and `wa_eco` CTEs as above
2. `obs_pt` CTE: `SELECT _dlt_id, id, user__login, observed_on, longitude, latitude, ST_Point(longitude, latitude) AS pt FROM inaturalist_data.observations WHERE longitude IS NOT NULL AND latitude IS NOT NULL`
3. `with_specimen` CTE: JOIN obs_pt to `inaturalist_data.observations__ofvs sc ON sc._dlt_root_id = op._dlt_id AND sc.field_id = 8338 AND sc.value != ''`, getting `CAST(sc.value AS INTEGER) AS specimen_count`. Also LEFT JOIN `observations__ofvs sid ON sid._dlt_root_id = op._dlt_id AND sid.field_id = 9963` for `TRY_CAST(sid.value AS INTEGER) AS sample_id`.
4. Same spatial join pattern (with_county, county_fallback, final_county, with_eco, eco_fallback, final_eco) but joining on `_dlt_id` instead of `occurrence_id`

**`export_counties_geojson(con)` function (GEO-01):**
```python
rows = con.execute("""
SELECT name AS NAME,
       ST_AsGeoJSON(ST_SimplifyPreserveTopology(ST_GeomFromText(geometry_wkt), 0.001))
FROM geographies.us_counties
WHERE state_fips='53'
""").fetchall()
features = [
    {"type": "Feature", "properties": {"NAME": name}, "geometry": json.loads(geom)}
    for name, geom in rows
]
out = ASSETS_DIR / "counties.geojson"
out.write_text(json.dumps({"type": "FeatureCollection", "features": features}))
print(f"  counties.geojson: {len(features)} features, {out.stat().st_size:,} bytes")
```
