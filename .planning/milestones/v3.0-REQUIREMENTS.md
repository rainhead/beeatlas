# Requirements: v3.0 Mapbox GL JS Migration (Archive)

## Milestone Goal

Replace OpenLayers with Mapbox GL JS v3 to gain WebGL rendering for 250K+ points, drop legacy mapping JS from the application bundle, and unlock 3D terrain, vector tiles, and geocoding for future milestones.

## Notes

This milestone was a platform migration with **no formal REQ IDs**. Functional requirements were expressed as per-phase **Success Criteria** (8 must-haves per phase, 24 total) and verified directly via `*-VERIFICATION.md` plus human UAT at the end of each phase. See `v3.0-ROADMAP.md` for the full per-phase success criteria and verification status.
