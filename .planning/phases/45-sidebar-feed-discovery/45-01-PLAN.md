**Step 3 — Add _computeActiveFeedEntries method** (place near _runFilterQuery):

```typescript
private _computeActiveFeedEntries(): void {
  const entries = this._filterState.selectedCollectors
    .map(c => c.recordedBy ? this._feedIndex.get(c.recordedBy) : undefined)
    .filter((e): e is FeedEntry => e !== undefined);
  this._activeFeedEntries = entries;
}
```

Key: uses `c.recordedBy` (not `c.displayName`) to match `filter_value` in index.json. Per Pitfall 1 in RESEARCH.md. Collectors with `recordedBy === null` (iNat-only) are skipped — they have no ecdysis collector feed.

fetch('/data/feeds/index.json')
