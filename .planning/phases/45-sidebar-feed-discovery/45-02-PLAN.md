```typescript
private _renderFeedsSection() {
  if (this.activeFeedEntries.length === 0) return nothing;
  return html`
    <div class="feeds-section">
      <h3 class="feeds-header">Feeds</h3>
      ${this.activeFeedEntries.map(entry => html`
        <div class="feed-row">
          <span class="feed-label">${entry.filter_value} \u2014 determinations</span>
          <span class="feed-actions">
            <button class="feed-copy-btn" @click=${() => navigator.clipboard.writeText(entry.url)}>Copy URL</button>
            <a href="${entry.url}" target="_blank" rel="noopener">Open Feed</a>
          </span>
        </div>
      `)}
    </div>
  `;
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| activeFeedEntries prop → DOM | FeedEntry fields rendered as text content and href attributes in Lit templates |
| user click → clipboard | navigator.clipboard.writeText with pipeline-generated URL |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-45-04 | Injection | feed URL in href attribute | mitigate | Lit html template auto-escapes attribute values; entry.url is root-relative path from pipeline — no user input. No innerHTML usage. |
| T-45-05 | Injection | filter_value in text content | mitigate | Rendered via Lit text interpolation (${entry.filter_value}), which auto-escapes. No innerHTML. |
| T-45-06 | Information Disclosure | clipboard write | accept | Copies a public URL to clipboard; no sensitive data. Clipboard API requires HTTPS (guaranteed by CloudFront). |
</threat_model>
