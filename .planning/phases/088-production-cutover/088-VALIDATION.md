## Sampling Rate

- **After every task commit:** `bash data/dbt/run.sh build` (dbt models + tests, ~3 s)
- **After every plan wave:** `bash data/dbt/run.sh build && cd data && uv run pytest data/tests/test_dbt_diff.py && cd .. && npm test`
- **Before `/gsd-verify-work`:** Full suite green + manual frontend smoke check (VALIDATE-02 acceptance)
- **Max feedback latency:** ~5 s (dbt build is the long pole; pytest + vitest each ~1 s)
