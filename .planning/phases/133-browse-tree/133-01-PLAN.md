<objective>

that `_pages/species.njk` walks at build time to render ~800 `<details>` nodes
(603 species + 191 higher-rank rows). This is the data foundation for the whole
phase — the template and client entry consume the shape produced here.

genus parent name carried on the node). Implements TREE-01 (default
family→genus→species with subfamily/tribe/subgenus available deeper),
TREE-02 (per-node count split), TREE-04 (bee-only sourcing + working node links).
Output: a new `fullTree` export (the existing placeholder `tree`/`buildTree`/
`TAXON_LEVELS` is replaced/hardened) and Wave 0 test coverage in
`data-species.test.ts`.
