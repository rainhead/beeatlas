  <what-built>URL state synchronization for the Washington Bee Atlas. Panning/zooming updates the URL query string in real time. Copying the URL and opening it restores the exact map view, active filters, and selected specimen. Browser back/forward navigate between settled states.</what-built>
  <how-to-verify>
Open the local dev server URL (reported by Task 1) in your browser, then verify each scenario:

**Scenario A — Default load (no URL params):**
1. Open http://localhost:5173 (or the reported URL) with no query string
2. EXPECT: Map centered on Washington State (roughly lon=-120.5, lat=47.5, zoom=7)
3. EXPECT: URL bar shows params like `?x=-120.5000&y=47.5000&z=7.00`

**Scenario B — Pan/zoom updates URL:**
1. Pan the map east and zoom in
2. EXPECT: URL bar updates with new x, y, z values while panning
3. EXPECT: After you stop moving (~500ms), the browser history gains a new entry (back button becomes active)

**Scenario C — URL round-trip (copy/paste):**
1. Pan to a specific location and note the URL
2. Copy the full URL from the address bar
3. Open a new tab and paste the URL
4. EXPECT: New tab opens to the exact same map position and zoom level

**Scenario D — Filter URL encoding:**
1. In the sidebar, select a taxon (e.g., type "Bombus" and pick "Bombus (genus)")
2. EXPECT: URL gains `taxon=Bombus&taxonRank=genus` params
3. Copy the URL, open a new tab
4. EXPECT: New tab loads with Bombus filter active — map shows ghosted clusters for non-Bombus, filter input shows "Bombus (genus)"

**Scenario E — Year filter URL:**
1. Set "From year" to 2023
2. EXPECT: URL gains `yr0=2023`
3. Copy URL, open new tab — EXPECT: year filter restored (From year input shows 2023)

**Scenario F — Browser back button:**
1. Pan map to location A, wait 1 second
2. Pan map to location B, wait 1 second
3. Press browser back button
4. EXPECT: Map returns to location A

**Scenario G — Selected occurrence URL (o param):**
1. Click a specimen cluster to open its detail panel
2. EXPECT: URL gains `o=ecdysis:{id}` param
3. Copy URL, open new tab
4. EXPECT: New tab loads with specimen detail panel open for that cluster

**If any scenario fails, describe the exact failure behavior.**
