**Bug 1 — Initial replaceState strips o=:**

In firstUpdated (line 420-421):
```typescript
const initParams = buildSearchParams(initCenter, view.getZoom()!, filterState, null);
window.history.replaceState({}, '', '?' + initParams.toString());
```
The fourth argument to buildSearchParams is `null`, so o= is never included in the initial replaceState.
Fix: pass `initialParams.occurrenceId` (or the first of parsed occurrence IDs) instead of `null`.

Additionally, _restoreSelectedOccurrence (called from the data-load callback) never calls _pushUrlState()
after restoring the selection, so even if o= is preserved initially, the sidebar opens but the URL
doesn't re-encode the selection. Fix: call _pushUrlState() after successful restore in the data-load callback.

**Bug 2 — Cluster click encodes only one occurrence:**

In the singleclick handler (line 492):
```typescript
this._selectedOccId = (toShow[0]!.getId() as string) ?? null;
```
Only the first occurrence ID is stored. toShow may have many features from a cluster.
Fix: Store all IDs and encode them comma-separated in the URL.

**Design decision for multi-occurrence o= encoding:**
Encode all occurrence IDs comma-separated: `o=ecdysis:123,ecdysis:456,ecdysis:789`
On restore, look up each ID individually and combine the features before calling buildSamples().
This exactly reconstructs what the user saw when they clicked the cluster.

**Interface changes required:**

1. ParsedParams: change `occurrenceId: string | null` to `occurrenceIds: string[]`
2. _selectedOccId: change from `string | null` to `string[] | null` (rename to _selectedOccIds)
3. buildSearchParams: accept `selectedOccIds: string[] | null` and join with comma
4. parseUrlParams: return occurrenceIds as string[] (split comma-separated value, validate prefix)
5. _restoreSelectedOccurrence: rename to _restoreSelectedOccurrences, accept string[]
6. All callers: update accordingly

**Key invariant:** Feature IDs are stored as 'ecdysis:{numeric_id}' (set in parquet.ts line 31).
The o= param stores these full namespaced IDs. specimenSource.getFeatureById('ecdysis:123') works.

```typescript
function buildSearchParams(
  center: number[],
  zoom: number,
  fs: typeof filterState,
  selectedOccIds: string[] | null
): URLSearchParams {
  const params = new URLSearchParams();
  params.set('x', center[0]!.toFixed(4));
  params.set('y', center[1]!.toFixed(4));
  params.set('z', zoom.toFixed(2));
  if (fs.taxonName !== null) {
    params.set('taxon', fs.taxonName);
    params.set('taxonRank', fs.taxonRank!);
  }
  if (fs.yearFrom !== null) params.set('yr0', String(fs.yearFrom));
  if (fs.yearTo   !== null) params.set('yr1', String(fs.yearTo));
  if (fs.months.size > 0)  params.set('months', [...fs.months].sort((a, b) => a - b).join(','));
  if (selectedOccIds !== null && selectedOccIds.length > 0) {
    params.set('o', selectedOccIds.join(','));
  }
  return params;
}
```

```typescript
private _restoreSelectedOccurrences(occIds: string[]) {
  const features: Feature[] = [];
  for (const occId of occIds) {
    const feature = specimenSource.getFeatureById(occId) as Feature | null;
    if (feature) features.push(feature);
  }
  if (features.length === 0) {
    this.selectedSamples = null;
    this._selectedOccIds = null;
    return;
  }
  const toShow = isFilterActive(filterState)
    ? features.filter(f => matchesFilter(f, filterState))
    : features;
  if (toShow.length > 0) {
    this.selectedSamples = buildSamples(toShow);
    this._selectedOccIds = toShow.map(f => f.getId() as string);
  } else {
    this.selectedSamples = null;
    this._selectedOccIds = null;
  }
}
```
