Add a helper function:
```python
def _read_prj(zip_path: Path, shp_stem: str) -> str:
    """Read the WKT CRS definition from a shapefile's .prj file inside a zip."""
    with zipfile.ZipFile(zip_path) as zf:
        return zf.read(f"{shp_stem}.prj").decode().strip()
```

Rewrite `load_geographies()` to:
1. Connect to DuckDB: `con = duckdb.connect(DB_PATH)`
2. Load spatial extension: `con.execute("INSTALL spatial; LOAD spatial;")`
3. Create schema: `con.execute("CREATE SCHEMA IF NOT EXISTS geographies")` (per D-10)
4. For each source, call `_download()` then use `CREATE OR REPLACE TABLE geographies.<name> AS SELECT ... FROM ST_Read(?)` (per D-08)
