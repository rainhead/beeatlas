```python
    """Add county and ecoregion_l3 columns to df using spatial join.

    Returns df.copy() with new 'county' (pd.StringDtype) and 'ecoregion_l3' (pd.StringDtype) columns.
    """
```

From 16-RESEARCH.md — two-step sjoin pattern (verified against live data):
- Step 1: gpd.sjoin(pts, gdf, how='left', predicate='within') — accurate for EPSG:4326
- After step 1: joined[~joined.index.duplicated(keep='first')] — deduplicate shared boundary hits
- Step 2 (for null rows only): reproject both to EPSG:32610, then gpd.sjoin_nearest — avoids geographic CRS warning
- County name column: 'NAME' (TIGER/Line); ecoregion name column: 'NA_L3NAME' (EPA shapefile)
- Output columns: 'county' = NAME values, 'ecoregion_l3' = NA_L3NAME values, both pd.StringDtype()
