**Setup:**

1. Ensure latest code is checked out and `npm run build` is green.
2. Start the dev server: `npm run dev` (or `npx serve _site`).
3. Open `http://localhost:8080/species/` in a fresh browser profile (no cached state).

**Use case 1 — "Which species of *Eucera* are present in this ecoregion?"**

1. In the taxon nav, expand Apidae → (subfamilies) → find the genus *Eucera*. Click *Eucera* to filter the cards.
2. In `<bee-species-filter>`, select an ecoregion-l3 value (try one with known Eucera presence; if unsure, try "Columbia Plateau" or "North Cascades").
3. Record:
   - The list of *Eucera* species shown (post-filter, non-muted cards).
   - For each, the visible "N records" badge value (filtered count).
4. Cross-check by running the same filter in the SPA (`/?taxon=Eucera&taxonRank=genus` plus the ecoregion filter). The species set should match.
5. Mark PASS if the species list is non-empty and matches the SPA cross-check; FAIL otherwise.

**Use case 2 — "Which are most likely / frequently collected?"**

1. Clear filters. In the taxon nav, optionally narrow to a family of interest (e.g., Apidae or Halictidae).
2. Visually scan the cards for the highest "N records" badges. Note the top 5 species + counts.
3. Cross-check the top species by sorting `public/data/species.json` by `occurrence_count` desc:
   ```bash
   node -e "const a=require('./public/data/species.json'); a.sort((x,y)=>y.occurrence_count-x.occurrence_count); console.log(a.slice(0,10).map(s=>({n:s.scientificName,c:s.occurrence_count})));"
   ```
4. Mark PASS if the page surfaces the same top species (order doesn't have to match exactly — visual scan is approximate); FAIL if obviously high-count species are missing or if the badges show stale numbers.

**Carry-in regression checks:**

5. **T2 layout:** Resize the window to < 768 px wide. Confirm the nav becomes a `<details>` block above the cards (single column). Resize to ≥ 768 px. Confirm the nav becomes a sticky left rail with cards on the right.
6. **T7 month-letter:** Find a card with a low-record species (n < 5) where occurrences are in a single month — e.g. find one via:
   ```bash
   node -e "const a=require('./public/data/species.json'); for (const s of a) { if (s.occurrence_count > 0 && s.occurrence_count < 5 && s.month_histogram && s.month_histogram.filter(n=>n>0).length === 1) { console.log(s.scientificName, s.occurrence_count, s.month_histogram); break; } }"
   ```
   Visit that card. Confirm the seasonality fallback reads "N record(s)" with NO trailing comma + month letter.

**PERF spot checks:**

7. PERF-01: `npm run build` final line shows "ok species-…: NN.N KB / 100.0 KB (MM.M KB headroom)". Record the value.
