## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline gaps between icon buttons |
| sm | 8px | Compact internal padding (e.g. between hamburger icon and edge) |
| md | 16px | Tab horizontal padding, default element spacing |
| lg | 24px | Header vertical rhythm, section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Touch targets for icon buttons (Map/Table): minimum 44px × 44px clickable area (WCAG 2.5.5)
- h1 margin: 16px top/bottom, 8px left (1rem/0.5rem) — existing value, preserved

Source: existing `.toggle-btn` horizontal padding `1rem` in bee-sidebar.ts; h1 style in index.css

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px (1rem) | 400 | 1.5 |
| Label / tab | 14.4px (0.9rem) | 500 | 1.2 |
| Heading (h1) | 19.2px (1.2rem) | 400 | 1.5 |

Two weights only: 400 (body and heading) and 500 (all tab labels — active and inactive alike). Active tab state is communicated via accent color (`color: var(--accent)`) and `border-bottom: 2px solid var(--accent)`, not font-weight change.

Source: `:root` in index.css (`font-size` 16px base, `line-height 1.5`, `font-weight 400`); `.toggle-btn` in bee-sidebar.ts (`font-size: 0.9rem; font-weight: 500`)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `rgb(8, 13, 38)` (`--header-bg`) | Header bar background |
| Secondary (30%) | `#fafafa` (`--surface`) | App surface behind header (not in header itself) |
| Accent (10%) | `#2c7a2c` (`--accent`) | Active nav tab underline + active tab text; active view icon |
| Inactive / hint | `#767676` (`--text-hint`) | Inactive nav tab text |
| Disabled | white at `opacity: 0.4` | Placeholder tabs (Species, Plants): `pointer-events: none` |
| Icon muted | white at `opacity: 0.8` | Inactive view icon buttons (matches `.github-link` pattern) |
| Icon active | white at `opacity: 1.0` + accent underline | Active view icon button |

Accent reserved for:
1. Active nav tab bottom border (`border-bottom: 2px solid var(--accent)`)
2. Active nav tab text color
3. Active view mode icon button (full opacity, distinct from muted inactive)

Destructive: none — no destructive actions in this phase.

Source: index.css `--header-bg`, `--accent`, `--text-hint`; `.toggle-btn.active` in bee-sidebar.ts; `.github-link` opacity pattern in index.css

## Interaction Contract

### Nav Tabs (left side of header)

| State | Visual |
|-------|--------|
| Active (Specimens or Samples) | `color: var(--accent)`, `border-bottom: 2px solid var(--accent)`, white text at full opacity, font-weight: 500 |
| Inactive | white text at opacity 0.7, no bottom border, font-weight: 500 |
| Hover (inactive) | white text at opacity 1.0, subtle background tint (`rgba(255,255,255,0.08)`) |
| Disabled (Species, Plants) | white text at `opacity: 0.4`, `pointer-events: none`, `cursor: default` |

Tab vertical padding: inherits from existing `.toggle-btn` (0.6rem / 9.6px at 16px base); pending normalization to 8px in Phase 53 when the sidebar is refactored.

### View Toggle Icons (right side of header)

| State | Visual |
|-------|--------|
| Active (Map or Table) | white at opacity 1.0 + `border-bottom: 2px solid var(--accent)` OR accent-colored fill |
| Inactive | white at opacity 0.6 |
| Hover (inactive) | white at opacity 0.9 |
| Touch target | 44px × 44px minimum |

### Hamburger Menu (≤640px viewport)

| State | Visual |
|-------|--------|
| Collapsed | `<summary>` shows ☰ icon (or equivalent 24×24 SVG), white, full opacity |
| Expanded | `<details>` open; nav items stack vertically below header using same tab styles; hamburger icon shows ✕ or remains ☰ (implementation choice) |
| Breakpoint | `@media (max-width: 640px)`: inline tabs hidden, `<details>` shown. Above 640px: inline tabs shown, `<details>` hidden. |

Source: CONTEXT.md D-06, D-07

## Layout Contract

```
┌─────────────────────────────────────────────────────────┐
│ [BeeAtlas h1]  [Specimens] [Samples] [Species] [Plants]  ··  [map-icon] [table-icon] [github-icon] │
└─────────────────────────────────────────────────────────┘
   ← left                                           right →

Mobile (≤640px):
┌──────────────────────────────────────┐
│ [BeeAtlas h1]          [☰]  [map] [table] [github] │
└──────────────────────────────────────┘
  [details open]:
  ┌──────────────────────────────────┐
  │ Specimens                        │
  │ Samples                          │
  │ Species (greyed)                 │
  │ Plants  (greyed)                 │
  └──────────────────────────────────┘
```

- Header is full-width, flex row, `justify-content: space-between`, `align-items: center`
- Left group: h1 title + nav tabs (or hamburger on mobile)
- Right group: view icon buttons + GitHub icon
- Header height: content-driven (no fixed px height); follows existing `h1` margin rhythm
- Primary focal point: the active nav tab (accent underline + full-opacity accent text) is the primary visual anchor; h1 establishes identity at rest.
