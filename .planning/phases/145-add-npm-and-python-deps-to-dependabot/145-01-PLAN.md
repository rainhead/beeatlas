<objective>
Extend `.github/dependabot.yml` so Dependabot version updates cover all three
dependency ecosystems in this repo — github-actions (already present), npm
(root `package.json`/`package-lock.json`), and Python via uv
(`data/pyproject.toml`/`data/uv.lock`) — with minor+patch updates grouped into
a single PR per ecosystem and all three on a weekly schedule.

Purpose: Keep dependencies current with reviewable, low-noise PRs. Grouping
minor+patch reduces PR churn while keeping major (potentially breaking) bumps
as isolated, individually-reviewable PRs.

Output: An updated `.github/dependabot.yml` (the only file this phase touches)
with three `updates:` entries, each weekly and each carrying a minor+patch
`groups:` block.
</objective>
