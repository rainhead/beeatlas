  <action>
Insert the following block into `infra/lib/beeatlas-stack.ts` AFTER the CloudFront invalidation policy statement (after line 143) and BEFORE the `// ── Outputs` comment block (before line 145):

```typescript
    // ── Pipeline Cache Bucket ─────────────────────────────────────────────
    // Separate bucket from siteBucket — keeps pipeline artifacts off the CDN-served prefix.
    // RETAIN on destroy: don't lose the cache if the stack is ever torn down.
    const cacheBucket = new s3.Bucket(this, 'PipelineCacheBucket', {
      blockPublicAccess: s3.BlockPublicAccess.BLOCK_ALL,
      removalPolicy: cdk.RemovalPolicy.RETAIN,
    });

    // Scoped access: GetObject + PutObject on cache/* prefix only (not s3:*)
    deployerRole.addToPolicy(new iam.PolicyStatement({
      effect: iam.Effect.ALLOW,
      actions: ['s3:GetObject', 's3:PutObject'],
      resources: [`${cacheBucket.bucketArn}/cache/*`],
    }));

    // ListBucket scoped to cache/* — needed to detect cache miss vs empty prefix
    deployerRole.addToPolicy(new iam.PolicyStatement({
      effect: iam.Effect.ALLOW,
      actions: ['s3:ListBucket'],
      resources: [cacheBucket.bucketArn],
      conditions: {
        StringLike: { 's3:prefix': ['cache/*'] },
      },
    }));
```

Also add the following CfnOutput inside the existing Outputs block (after the `DeployerRoleArn` output):

```typescript
    new cdk.CfnOutput(this, 'CacheBucketName', {
      value: cacheBucket.bucketName,
      description: 'Pipeline cache S3 bucket → GitHub var CACHE_BUCKET_NAME',
    });
```
