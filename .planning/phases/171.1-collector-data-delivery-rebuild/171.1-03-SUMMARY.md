## Decisions Made

- **Fine-grained PAT scope:** beeatlas repo only, Contents: Read and write — minimum for `POST /repos/{owner}/{repo}/dispatches`. Rationale: T-171.1-06 elevation-of-privilege mitigation.
- **PAT storage:** Root-only file `~/.secrets/beeatlas-github-pat` (mode 600) on maderas, outside the repo tree, never committed. Rationale: T-171.1-07 information-disclosure mitigation.
- **Non-fatal dispatch:** Failure prints `WARN: repository_dispatch failed (non-fatal)` to stderr; nightly exits 0. Rationale: S3 data is already correct; stale pages are recoverable.
- **curl over gh:** `gh` availability on maderas was unverified; `curl` is guaranteed on any Linux host.
