```yaml

    branches: ['**']

    branches: ['**']
```

## Shared Patterns

### Content-hash + S3 upload
**Source:** `data/nightly.sh` lines 273–282 (`_upload_hashed` function)
**Apply to:** both new artifact upload lines in nightly.sh
```bash
_upload_hashed() {
    local src="$1" basename="$2"; shift 2
    local ext="${src##*.}"
    local hash; hash=$(sha256sum "$src" | awk '{print $1}' | cut -c1-12)
    local hashed_name="${basename}-${hash}.${ext}"
    aws --profile "$AWS_PROFILE" s3 cp --no-progress \
        --cache-control "public, max-age=31536000, immutable" \
        "$@" "$src" "s3://$BUCKET/data/$hashed_name" >&2
    echo "$hashed_name"
}
```

### manifest.json → jq → aws s3 cp resolution
**Source:** `.github/workflows/deploy.yml` lines 44–49
**Apply to:** the two new collector fetch lines in deploy.yml
```yaml
SPECIES_FILE=$(jq -r .species /tmp/manifest.json)
aws s3 cp s3://${{ vars.S3_BUCKET_NAME }}/data/$SPECIES_FILE public/data/species.json
```
Pattern: one `jq -r .<key>` to get the hashed filename, one `aws s3 cp` to fetch it into `public/data/<stable-name>.json`.
