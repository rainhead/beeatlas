## Decisions Made

- **PLAIN over gzip for both artifacts (D-08):** `aws s3 cp` downloads raw bytes; a Content-Encoding:gzip S3 object causes `readFileSync`+`JSON.parse` to fail at build time. `_upload_hashed` (plain) is the correct pattern. CloudFront auto-compresses JSON responses for browsers so there is no bandwidth penalty.
- **collectors in LOCAL_NAMES, collector_event_pages in INTENTIONALLY_SKIPPED:** collectors.json is ~3.5 MB and appropriate for daily baseline diffs (mirrors `species.json`). collector_event_pages.json is ~29 MB — too large for daily pull (mirrors `occurrences_db`).
