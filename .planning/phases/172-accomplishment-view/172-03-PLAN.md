<objective>
Create `data/collector_maps.py` — a new SVG generator that emits two binary-fill
coverage maps per WABA collector:
- `collector-maps/{login}.svg` — WA counties the collector contributed to
  (ACCOM-01 / D-02),
- `collector-maps/{login}-eco.svg` — EPA Level III ecoregions the collector
  contributed to (ACCOM-03 / D-03).

polygons from `ecoregions.geojson` (keyed by `NA_L3NAME`), the shared
`_project`/`_ring_to_path` projection, the `.county`/`.checklist-county`
`STYLE_CSS` classing, deepcopy-per-collector backdrop, attrib-sort determinism,
and wipe-and-rewrite idempotency. NO occurrence dots, NO choropleth (D-02).

This plan turns the RED `test_collector_maps.py` tests from Plan 01 GREEN.

Purpose: pre-generate static coverage SVGs (D-06) delivered via the stable-URL
S3 path (wired in Plan 04 — same mechanism as species-maps/).
Output: `data/collector_maps.py` with `generate_collector_maps_step`.
</objective>
