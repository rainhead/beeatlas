  <files>src/filter.ts</files>
  <action>
    Add the following to src/filter.ts after the existing exports (before the end of the file).

    1. Add three interface exports. Copy the exact definitions from bee-sidebar.ts — do NOT import
       from bee-sidebar.ts. These are standalone interface declarations that belong in filter.ts:

       - DataSummary: { totalSpecimens, speciesCount, genusCount, familyCount, earliestYear, latestYear } (all number)
       - TaxonOption: { label: string; name: string; rank: 'family' | 'genus' | 'species' }
       - FilterChangedEvent: same fields as FilterState but with additional elevMin/elevMax/selectedPlace
         (matches current FilterChangedEvent in bee-sidebar.ts exactly — do NOT simplify)

    2. Add export async function queryListPage:

       Signature: queryListPage(
         f: FilterState,
         page: number,
         sortBy: SpecimenSortBy = 'date',
         selectedEcdysisIds: number[] = [],
         selectedInatIds: number[] = [],
         selectionBounds: { west: number; south: number; east: number; north: number } | null = null
       ): Promise<{ rows: OccurrenceRow[]; total: number }>

       Implementation pattern (mirrors queryTablePage but uses WHERE intersection, not ORDER priority):
       - Call buildFilterSQL(f) to get occurrenceWhere
       - Build selParts: if selectedEcdysisIds.length > 0, add "ecdysis_id IN (...)"
         if selectedInatIds.length > 0, add "observation_id IN (...)"
       - Build boundsClause: if selectionBounds !== null, add
         " AND lat BETWEEN ${south} AND ${north} AND lon BETWEEN ${west} AND ${east}"
       - selFilter: if selParts.length > 0, " AND (${selParts.join(' OR ')})" else ""
       - fullWhere = "(${occurrenceWhere})${selFilter}${boundsClause}"
       - orderBy: use SPECIMEN_ORDER_MODIFIED when sortBy === 'modified', else SPECIMEN_ORDER
       - offset = (page - 1) * PAGE_SIZE
       - await tablesReady; const { sqlite3, db } = await getDB();
       - COUNT(*) query on fullWhere → total
       - SELECT OCCURRENCE_COLUMNS.join(', ') FROM occurrences WHERE fullWhere ORDER BY orderBy LIMIT PAGE_SIZE OFFSET offset → rows
       - return { rows: rows as unknown as OccurrenceRow[], total }

       The IDs in selectedEcdysisIds and selectedInatIds are pre-validated integers (same
       contract as queryTablePage). Join them directly with .join(',') as integers — no quoting.

       Note: SPECIMEN_ORDER and SPECIMEN_ORDER_MODIFIED are module-level constants already in
       filter.ts. PAGE_SIZE = 100 is also already there. Use them.
  </action>
