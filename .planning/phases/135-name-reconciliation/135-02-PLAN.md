<objective>
Build the tiered checklist name resolver as a standalone module `data/resolve_checklist_names.py`, mirroring `resolve_taxon_ids.py`. It is a no-op on the nightly path and only does work under `--refresh-checklist`: it normalizes verbatim names, resolves slash-compounds to their lowest-common-ancestor taxon_id from taxa.csv.gz, runs one-time GBIF backbone lookups, generates rapidfuzz candidates for the curator-review CSV, writes the committed audit CSV + the GBIF synonym seed, and exposes a build-blocking gate that fails only on no-match-anywhere.

Purpose: RCN-02/03/04/05 — the resolution layer + offline-reproducible committed cache + curator surfaces, with zero nightly network calls.
Output: 1 new module + 3 committed CSV artifacts (audit, fuzzy-review, gbif seed).
</objective>
