Collection events (group outings, typically scheduled via Canvas) are a central part of how WABA volunteers coordinate. BeeAtlas is the natural home for:

- Scheduling and advertising upcoming events
- Sharing field photos afterwards
- Surfacing iNaturalist comments and determinations in context
- Showing recent activity to convey liveness and togetherness

**Why deferred:** Cold-start problem — a half-built community platform feels emptier than no community features at all. Build this once there are people using the site semi-regularly.

**Starting point:** Surface iNat comments on specimens/samples before building any on-site interaction. This requires no new content creation from users and conveys community presence immediately.
