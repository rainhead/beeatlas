Welcome! This is a nascent (March 2026) project to build tools to support volunteer coordination by bee atlas projects. It currently has only [Washington Bee Atlas](https://agr.wa.gov/departments/insects-pests-and-weeds/insects/apiary-pollinators/pollinator-health/bee-atlas/) data.

Sorry, there isn't much documentation yet. Feel free to reach out by filing an issue or writing rainhead@gmail.com.

Data sources:
- [Ecdysis](https://ecdysis.org/collections/list.php?datasetid=44)
- [iNaturalist](https://www.inaturalist.org/projects/washington-bee-atlas-waba-plant-images-sample-ids-1854c0dc-0780-41e9-93f7-1f582b4df096)
